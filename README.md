## What I'm working on

A system that understands the difference between a news article about violence and someone inciting it. Targeting the gap that costs platforms hundreds of millions a year.

**vLLM-Omni contributions** — actively contributing to [vllm-project/vllm-omni](https://github.com/vllm-project/vllm-omni), the open-source omnimodal inference engine. Working on engine reliability and codebase correctness.

**ML systems** — studying how large-scale inference systems are actually built: memory management, scheduling, quantization, multimodal pipelines. Reading papers, reading code, building things to understand them.
